# Web Design (ProInterview — frontend)

This is a code bundle for Web Design. The original project is available at https://www.figma.com/design/XACL19kt9UZ2xQwTZsUDMq/Web-Design.

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server (Vite; proxy `/api` → backend `:5000` khi chạy kèm backend).
